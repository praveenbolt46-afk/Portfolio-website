Portfolio Website

This is a simple two-page portfolio website built using HTML and basic CSS. The website includes a Home page to introduce myself and highlight my skills, and a Contact page that allows visitors to reach out to me. The project focuses on clean structure, basic styling, and beginner-friendly web development practices.

Pages

Home – Introduction and overview of my profile

Contact – Contact details and/or contact form

Technologies Used

HTML5

CSS3

Purpose

This project was created to practice basic front-end development concepts, including page navigation, layout design, and styling using CSS.
